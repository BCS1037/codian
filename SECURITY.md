# Security policy

## Reporting a vulnerability

Do not open a public issue for a suspected vulnerability.

Use [GitHub private vulnerability reporting](https://github.com/BCS1037/codian/security/advisories/new). Include affected version or commit, reproduction steps, impact, and any suggested mitigation.

Do not include real credentials, private vault content, transcripts, or personal data. Use synthetic fixtures.

## Supported versions

Codian has not published a public binary release. Security fixes currently target the default branch.
